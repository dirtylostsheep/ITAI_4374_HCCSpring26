## Project Overview
This project applies neuroscience-inspired concepts to artificial intelligence systems. It focuses on modeling and analyzing biologically inspired mechanisms such as learning, perception, or decision-making.

- Build or simulate a brain-inspired model
- Analyze system behavior and performance

- Attention / Decision-Making
